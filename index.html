AI JOB RADAR

隐私政策

AI 求职雷达 · Microsoft Edge 侧边助手

生效日期：2026 年 7 月 23 日

概述

AI 求职雷达（下称“本扩展”）帮助用户在 Microsoft Edge 中分析 BOSS 招聘岗位。本政策说明本扩展访问、保存和传输的数据。

访问的数据

  • 用户主动上传或粘贴的简历文字；
  • 用户在 BOSS 当前页面已加载的岗位名称、公司、岗位描述和链接；
  • 用户确认的学历、经验、技能、项目与经历画像；
  • 用户自行填写的模型服务商、Base URL、模型名称和 API Key；
  • 本扩展生成的岗位分析结果与沟通草稿。

使用目的

上述数据仅用于提取可核验的候选人画像、对照岗位要求、生成求职行动建议，以及生成用户主动请求的沟通草稿。本扩展不将数据用于广告、数据出售、跨站跟踪或与求职辅助无关的用途。

本地保存

API Key、简历原文、候选人画像和分析结果保存在 Microsoft Edge 提供的本地扩展存储中。本扩展开发者不运营用于保存这些数据的后台服务器，也不会收到这些数据。

第三方模型服务

首次连接模型前，本扩展会明确提示并请求用户主动同意。用户同意并发起操作后，简历文字、当前岗位描述和已确认画像会由 Microsoft Edge 直接发送至用户选择的模型服务商，例如 DeepSeek、OpenAI、用户填写的 OpenAI 兼容服务或本地 Ollama，仅用于返回本次分析或草稿。API Key 仅发送至用户配置的对应接口以完成身份验证。

第三方模型服务如何处理数据受其各自隐私政策与服务条款约束。用户应在使用前自行查看对应服务商政策。除本地 Ollama 外，远程接口必须使用 HTTPS。

用户控制

用户可以不勾选数据处理同意项，此时本扩展不会调用模型。用户可随时在“设置”中点击“清除 API Key、简历和全部分析记录”，立即删除本扩展在 Microsoft Edge 本地保存的数据并停止后续传输；也可以直接卸载本扩展。

数据出售、广告与分析

本扩展不出售、出租或交易用户数据，不包含广告，不使用第三方行为分析或跟踪 SDK。

数据安全与保留

本扩展只申请完成功能所必需的页面与接口权限。远程模型数据通过 HTTPS 传输，本地 Ollama 仅允许 localhost 或 127.0.0.1。数据在用户清除或卸载前保留在浏览器本地；第三方模型服务的保留规则由其各自政策决定。

儿童隐私

本扩展面向求职者，不以儿童为目标用户。

政策更新与联系

如政策发生实质变化，将随扩展版本更新并在商店说明中披露。隐私或数据删除问题可通过 GitHub Issues 联系发布者。
